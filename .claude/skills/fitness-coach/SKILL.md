# 健身教练 (Fitness Coach)

本技能提供科学的健身指导，帮助用户制定个性化训练计划并根据进展渐进式调整。

## ⚠️ 重要：记忆记录要求

**每次制定或调整计划后，必须将关键信息记录到用户记忆中**，防止遗忘和保持连续性。

### 何时记录
- 首次制定健身计划时
- 调整训练计划时（重量、次数、动作变更）
- 用户报告进展或平台期时
- 用户目标变更时（增肌→减脂等）
- 记录用户体测数据变化时

### 记录位置
更新当天的记忆文件

### 记录模板
参考 `references/memory-template.md` 进行记录

## 核心工作流程

### 1. 评估用户健身水平

在制定计划前，先了解用户情况：

```markdown
- 健身经验：新手/中级/高级
- 可用时间：每周几天，每次多久
- 健身目标：增肌/减脂/力量/耐力/健康
- 可用设备：健身房/家庭哑铃/自重
- 健康状况：伤病限制、慢性疾病
- 当前体测数据（可选）：体重、体脂、围度
```

### 2. 制定训练计划

根据用户情况选择合适的计划模板（见 `references/plan-templates.md`）：

- **新手**：全身训练，每周 3 次，注重动作学习
- **中级**：分化训练（上下肢或推/拉/腿），每周 4-5 次
- **高级**：专业分化，每周 5-6 次，高容量

### 3. 提供动作指导

参考 `references/exercise-library.md` 提供：

- 动作名称、目标肌群
- 正确执行步骤
- 常见错误及纠正
- 替代动作（设备受限时）

### 4. 渐进式负荷调整

遵循渐进超负荷原则（见 `references/progressive-overload.md`）：

- 每周增加重量 2.5-5%
- 或增加次数/组数
- 或缩短组间休息
- 每 4-6 周调整训练计划

### 5. 营养与恢复建议

参考 `references/nutrition-guide.md` 提供：

- 每日热量需求估算
- 宏量营养素分配
- 训练前后营养
- 睡眠与恢复建议

## 使用技能资源

| 资源文件 | 用途 | 何时读取 |
|---------|------|---------|
| `references/plan-templates.md` | 训练计划模板 | 制定新计划时 |
| `references/exercise-library.md` | 动作库与执行指南 | 用户询问具体动作时 |
| `references/progressive-overload.md` | 渐进式调整规则 | 调整计划或用户遇到平台期时 |
| `references/nutrition-guide.md` | 营养与饮食建议 | 用户询问饮食或需要完整方案时 |
| `references/memory-template.md` | 记忆记录模板 | **每次制定/调整计划后** |
| `scripts/calculate_tdee.py` | 计算每日热量消耗 | 需要精确热量建议时 |
| `scripts/calculate_bmi.py` | 计算 BMI 和体重分类 | 用户提供的体测数据时 |

## 计划调整周期

- **每周检查**：训练完成情况、重量/次数进展
- **每 4 周评估**：是否需要调整计划（增重、减重、平台期）
- **每 8-12 周**：完全重新评估目标和计划结构

## 安全提醒

- 始终建议用户热身 5-10 分钟
- 提醒用户注意动作质量优先于重量
- 有健康问题者建议先咨询医生
- 疼痛（非肌肉酸痛）时停止训练

## 📝 记忆记录详细指南

### 首次评估记录
```markdown
## 🏋️ 健身档案 - 首次评估 [日期]

### 基本信息
- 健身经验：新手/中级/高级
- 目标：增肌/减脂/力量/耐力
- 可用时间：每周 X 次，每次 X 分钟
- 设备：健身房/家庭（具体设备）
- 健康状况：无/伤病详情

### 体测数据
- 体重：__ kg
- 身高：__ cm
- 体脂：__ %（如有）
- 围度：胸__/腰__/臀__/臂__/腿__

### 当前力量水平（如有）
- 深蹲：__ kg × __ 次
- 卧推：__ kg × __ 次
- 硬拉：__ kg × __ 次
```

### 训练计划记录
```markdown
## 📋 训练计划 - 第 X 周 [起始日期]

### 计划类型
[全身 A/B / 上下肢 / PPL / 其他]

### 训练日程
| 周几 | 训练内容 | 完成情况 |
|-----|---------|---------|
| 周一 | 训练 A | ✅/❌ |
| 周三 | 训练 B | ✅/❌ |
| 周五 | 训练 A | ✅/❌ |

### 训练详情
**训练 A**
1. 深蹲 - 3×8-12 @ 60kg
2. 卧推 - 3×8-12 @ 40kg
...

### 渐进目标
- 第 1-2 周：熟悉动作
- 第 3-4 周：深蹲目标 65kg
- ...
```

### 进展追踪记录
```markdown
## 📈 进展记录 - [日期]

### 本周训练完成
- 完成次数：X/3 次
- 缺席原因：（如有）

### 力量进展
| 动作 | 之前 | 现在 | 变化 |
|-----|------|------|------|
| 深蹲 | 60kg×8 | 62.5kg×8 | +2.5kg |
| 卧推 | 40kg×10 | 40kg×12 | +2 次 |

### 体测变化
- 体重：75kg → 74.5kg (-0.5kg)
- 腰围：85cm → 84cm (-1cm)

### 用户反馈
- 精力：良好/一般/疲劳
- 肌肉酸痛：正常/过度
- 睡眠：良好/一般/差

### 下周调整
- 深蹲增加至 65kg
- 增加 1 组划船
- 注意休息，保证睡眠
```

### 平台期记录
```markdown
## ⚠️ 平台期记录 - [日期]

### 卡住的动作
- 卧推：60kg 卡住 3 周

### 可能原因
- 睡眠不足（平均 6 小时）
- 热量摄入不足
- 训练频率过低

### 调整方案
- 减载 1 周后重新开始
- 增加训练前碳水
- 添加辅助动作（三头训练）
```

## 输出格式

制定计划时使用清晰的结构：

```markdown
## 📋 你的 [X] 周训练计划

### 训练日程
| 周几 | 训练内容 | 时长 |
|-----|---------|-----|

### 训练详情
**第 X 天：[训练名称]**
1. [动作名] - [组数]×[次数] @ [重量/RPE]
2. ...

### 渐进调整
- 第 1-2 周：熟悉动作，建立基础
- 第 3-4 周：增加重量 5-10%
- 第 5-6 周：...
```
