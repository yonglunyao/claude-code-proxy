# 营养与饮食指南 (Nutrition Guide)

## 热量需求计算

### TDEE (每日总能量消耗) 估算

**公式**：TDEE = BMR × 活动系数

**BMR 计算 (Mifflin-St Jeor 公式)**：
- 男性：BMR = 10×体重 (kg) + 6.25×身高 (cm) - 5×年龄 + 5
- 女性：BMR = 10×体重 (kg) + 6.25×身高 (cm) - 5×年龄 - 161

**活动系数**：
| 活动水平 | 系数 | 描述 |
|---------|------|------|
| 久坐 | 1.2 | 几乎不运动，办公室工作 |
| 轻度活动 | 1.375 | 每周 1-3 次轻度运动 |
| 中度活动 | 1.55 | 每周 3-5 次中等运动 |
| 高度活动 | 1.725 | 每周 6-7 次高强度运动 |
| 极高活动 | 1.9 | 体力劳动 + 每日训练 |

### 目标热量调整

| 目标 | 热量调整 | 预期速度 |
|-----|---------|---------|
| 快速减脂 | TDEE - 500~750kcal | 每周 0.5-0.75kg |
| 温和减脂 | TDEE - 250~500kcal | 每周 0.25-0.5kg |
| 维持 | TDEE | 体重稳定 |
| 缓慢增肌 | TDEE + 250~300kcal | 每月 0.5-1kg |
| 快速增肌 | TDEE + 400~500kcal | 每月 1-2kg (含脂肪) |

## 宏量营养素分配

### 蛋白质
- **减脂期**：2.0-2.4g/kg 体重
- **增肌期**：1.6-2.2g/kg 体重
- **维持期**：1.4-1.8g/kg 体重

**优质来源**：鸡胸肉、鱼、牛肉、鸡蛋、乳清蛋白、希腊酸奶、豆腐

### 碳水化合物
- **训练日**：3-5g/kg 体重
- **休息日**：2-3g/kg 体重
- **减脂期**：2-3g/kg 体重
- **增肌期**：4-6g/kg 体重

**优质来源**：米饭、燕麦、土豆、红薯、全麦面包、水果

### 脂肪
- **推荐范围**：0.8-1.2g/kg 体重
- **最低不低于**：0.6g/kg (激素健康)

**优质来源**：橄榄油、牛油果、坚果、鱼油、蛋黄

## 宏量营养素计算示例

**用户**：男性，75kg，增肌期

```
蛋白质：75 × 2.0 = 150g (600kcal)
脂肪：75 × 1.0 = 75g (675kcal)
碳水化合物：剩余热量分配

假设 TDEE = 2500kcal，增肌 +300kcal = 2800kcal
碳水热量 = 2800 - 600 - 675 = 1525kcal
碳水化合物 = 1525 ÷ 4 = 381g

最终：150g 蛋白质 / 381g 碳水 / 75g 脂肪
```

## 训练前后营养

### 训练前 (1-3 小时)
- **碳水**：30-60g (提供能量)
- **蛋白质**：20-30g (预防肌肉分解)
- **脂肪**：少量 (延缓消化)

**示例**：
- 香蕉 + 乳清蛋白
- 燕麦 + 鸡蛋
- 米饭 + 鸡胸肉

### 训练后 (0-2 小时内)
- **蛋白质**：25-40g (肌肉合成)
- **碳水**：40-80g (补充糖原)
- **脂肪**：适量

**示例**：
- 乳清蛋白 + 香蕉
- 鸡胸肉 + 米饭
- 希腊酸奶 + 水果 + 蜂蜜

### 窗口期真相
- 训练后"合成窗口"比想象中宽（24-48 小时）
- 关键是全天总摄入，不是精确时机
- 但训练前后营养有助于表现和恢复

## 补剂建议

### 强烈推荐
| 补剂 | 剂量 | 作用 |
|-----|------|------|
| 乳清蛋白 | 25-50g/天 | 方便补充蛋白质 |
| 肌酸一水合物 | 3-5g/天 | 力量提升，肌肉饱满 |
| 咖啡因 | 3-6mg/kg | 训练前，提升表现 |

### 可考虑
| 补剂 | 剂量 | 作用 |
|-----|------|------|
| β-丙氨酸 | 3-5g/天 | 耐力提升 |
| 鱼油 | 1-3g/天 | 心血管健康，抗炎 |
| 维生素 D | 1000-4000IU/天 | 骨骼健康，免疫功能 |
| 复合维生素 | 按说明 | 填补营养缺口 |

### 不推荐 (性价比低)
- BCAA (不如直接吃蛋白质)
- 左旋肉碱 (效果有限)
- 睾酮促进剂 (基本无效)
- 大多数"燃脂"补剂

## 饮食执行技巧

### 备餐建议
1. **批量烹饪**：周末准备 3-4 天的蛋白质和主食
2. **分装储存**：按每餐分量分装，方便取用
3. **简单为主**：不需要复杂食谱，可持续最重要

### 外食策略
- 选择烤/蒸/煮，避免油炸
- 要求酱料分开
- 优先蛋白质 + 蔬菜，主食减半
- 火锅选清汤，多菜少加工品

### 应酬应对
- 提前吃点蛋白质，避免过度饥饿
- 优先选择瘦肉、海鲜
- 酒精适量（7kcal/g，影响恢复）
- 第二天回归正常饮食，不要补偿性节食

## 水分摄入

- **基础推荐**：30-40ml/kg 体重
- **训练日**：额外补充 500-1000ml
- **判断标准**：尿液淡黄色为宜
- **训练期间**：每 15-20 分钟补充 150-250ml

## 常见问题

### Q: 需要精确计算每餐吗？
A: 新手可以先估算，养成习惯后再精确。用手掌法估算：
- 蛋白质：1 手掌/餐
- 蔬菜：1 拳头/餐
- 碳水：1 杯状手/餐
- 脂肪：1 拇指/餐

### Q: 平台期如何调整饮食？
A: 
- 减脂平台：减少 100-200kcal 或增加有氧
- 增肌平台：增加 100-200kcal 或检查训练

### Q: 可以 cheat meal 吗？
A: 可以，建议：
- 频率：每周 1 次，不是 cheat day
- 控制量：比平时多 500-1000kcal 即可
- 心态：享受食物，不要有罪恶感

### Q: 素食者如何保证蛋白质？
A: 
- 豆类 + 谷物搭配（完全蛋白）
- 豆腐、天贝、面筋
- 植物蛋白粉
- 可能需要更高总量（消化率较低）

## 监控与调整

### 每周检查
- 体重（晨起空腹，周平均值）
- 围度（胸、腰、臀、臂、腿）
- 训练表现（重量、次数）
- 照片（同光线同角度）

### 调整时机
- **减脂**：2 周无进展 → 减少 100-200kcal
- **增肌**：1 月体重无增 → 增加 100-200kcal
- **表现下降**：检查热量、睡眠、压力
