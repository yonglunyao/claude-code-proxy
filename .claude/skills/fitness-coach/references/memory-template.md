# 健身记忆记录模板 (Memory Templates)

**重要**：每次制定或调整健身计划后，必须将关键信息记录到记忆文件中，防止遗忘。

---

## 文件位置

- 更新当天已有的记忆文件
- 用户健身档案记录到长期记忆

---

### 模板 4：平台期记录

```markdown
## ⚠️ 平台期记录 - [YYYY-MM-DD]

### 卡住的动作
| 动作 | 当前水平 | 卡住时长 | 上次进展 |
|-----|---------|---------|---------|
| 卧推 | 60kg×5 | 3 周 | 3 周前 57.5kg→60kg |
| 深蹲 | 80kg×5 | 2 周 | 2 周前 77.5kg→80kg |

### 可能原因分析
**训练因素**：
- [ ] 训练容量不足
- [ ] 训练频率过低
- [ ] 动作技术问题
- [x] 每次都力竭，恢复不足

**恢复因素**：
- [x] 睡眠不足（平均 6 小时/天）
- [ ] 热量摄入不足
- [x] 压力水平高（工作繁忙）
- [ ] 过度有氧

**其他因素**：
- 出差频繁，训练不规律
- 饮食不规律，有时 skip 训练餐

### 调整方案

**方案 A：减载周**
- 本周训练量减少 50%
- 重量减少 40%
- 专注动作质量
- 保证睡眠 7-8 小时

**方案 B：变换动作**
- 卧推 → 暂停卧推（改善底部力量）
- 添加辅助动作：三头下压、上斜哑铃

**方案 C：调整频率**
- 从每周 1 次推日 → 2 次（1 次重，1 次轻技术）

**选择方案**：A + B 结合
- 先减载 1 周
- 之后加入暂停卧推

### 执行计划
```
第 1 周（减载）：
- 卧推：40kg×5×3 组（轻，专注技术）
- 深蹲：50kg×5×3 组

第 2 周（恢复）：
- 卧推：55kg×5×4 组
- 添加暂停卧推：50kg×3×3 组

第 3 周（突破）：
- 卧推：62.5kg×5×4 组（新重量！）
```

### 跟进日期
- 下次评估：YYYY-MM-DD（2 周后）
```

## 📌 记录最佳实践

### 必须记录的时刻
1. ✅ 首次制定计划时（建立基线）
2. ✅ 每次调整重量/次数时
3. ✅ 每周结束时（进展总结）
4. ✅ 遇到平台期时（分析原因）
5. ✅ 目标变更时（记录成果和新方向）
6. ✅ 体测数据更新时

### 记录频率建议
- **每日**：训练日志（可选，用户自己记录）
- **每周**：进展总结（必须）
- **每月**：全面复盘（必须）
- **事件驱动**：计划调整、平台期、目标变更（必须）

### 记录要点
- **简洁**：用表格和列表，不要长段落
- **数据化**：重量、次数、体重、围度都要数字
- **可追踪**：便于对比前后变化
- **可执行**：包含明确的下一步调整
