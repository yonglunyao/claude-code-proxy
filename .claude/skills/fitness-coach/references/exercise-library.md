# 动作库 (Exercise Library)

## 胸部动作

### 卧推 (Bench Press)
**目标肌群**：胸大肌、三角肌前束、肱三头肌

**执行步骤**：
1. 仰卧，肩胛骨收紧下沉，双脚踩实地面
2. 握距略宽于肩，手腕保持中立
3. 吸气下放至胸部中下位置，肘部约 45°外展
4. 呼气推起至手臂伸直（不锁死）

**常见错误**：
- 耸肩 → 保持肩胛收紧
- 腰部过度拱起 → 收紧核心
- 弹胸 → 控制下放速度

**替代动作**：哑铃卧推、俯卧撑、器械推胸

---

### 俯卧撑 (Push-up)
**目标肌群**：胸大肌、三角肌前束、肱三头肌、核心

**执行步骤**：
1. 双手略宽于肩，身体呈直线
2. 核心收紧，臀部不塌陷不上翘
3. 下放至胸部接近地面
4. 推起至手臂伸直

**进阶变化**：
- 简单：跪姿俯卧撑、上斜俯卧撑
- 困难：钻石俯卧撑、弓箭手俯卧撑、击掌俯卧撑

---

## 背部动作

### 引体向上 (Pull-up)
**目标肌群**：背阔肌、大圆肌、肱二头肌

**执行步骤**：
1. 正手握杆，略宽于肩
2. 肩胛下沉启动，不要耸肩
3. 拉至下巴过杆，胸部尽量靠近
4. 控制下放至手臂伸直

**常见错误**：
- 摆动借力 → 保持身体稳定
- 半程动作 → 全程活动范围
- 耸肩 → 先下沉肩胛

**退阶**：弹力带辅助、离心训练、高位下拉
**进阶**：负重引体、单臂引体辅助

---

### 杠铃划船 (Barbell Row)
**目标肌群**：背阔肌、斜方肌、菱形肌、后束

**执行步骤**：
1. 俯身约 45°，膝盖微屈，背部挺直
2. 正手握杆，略宽于肩
3. 拉向腹部，肘部贴近身体
4. 控制下放

**常见错误**：
- 弯腰 → 保持脊柱中立
- 拉向胸部 → 应拉向腹部
- 过度后仰 → 核心收紧

---

## 腿部动作

### 深蹲 (Squat)
**目标肌群**：股四头肌、臀大肌、腘绳肌、核心

**执行步骤**：
1. 站距与肩同宽或略宽，脚尖微外展
2. 杠铃置于斜方肌上（高杠）或肩胛骨上（低杠）
3. 吸气，屈髋屈膝下蹲
4. 蹲至大腿平行或更低
5. 呼气站起，膝盖对准脚尖

**常见错误**：
- 膝盖内扣 → 主动外展膝盖
- 脚后跟离地 → 改善踝关节活动度
- 弯腰 → 收紧核心，胸部抬起

**替代动作**：腿举、高脚杯深蹲、前蹲

---

### 硬拉 (Deadlift)
**目标肌群**：腘绳肌、臀大肌、下背部、握力

**执行步骤**：
1. 站距与髋同宽，杠铃在足中上方
2. 屈髋屈膝，双手握杆
3. 收紧背部，胸部抬起
4. 蹬地站起，杠铃贴腿上行
5. 锁定髋部，不要过度后仰

**常见错误**：
- 圆背 → 保持脊柱中立
- 杠铃远离身体 → 贴腿上拉
- 先用膝盖 → 先蹬地伸髋

**变式**：罗马尼亚硬拉（侧重腘绳肌）、相扑硬拉

---

### 弓步蹲 (Lunge)
**目标肌群**：股四头肌、臀大肌、核心稳定性

**执行步骤**：
1. 站立，双手叉腰或持哑铃
2. 向前迈一大步下蹲
3. 前后膝盖均约 90°
4. 前腿发力回到起始位置

**常见错误**：
- 步幅太小 → 加大步幅
- 前膝超过脚尖太多 → 调整步幅
- 身体前倾过多 → 保持躯干直立

---

## 肩部动作

### 肩推 (Overhead Press)
**目标肌群**：三角肌前中束、肱三头肌、上胸

**执行步骤**：
1. 站姿或坐姿，核心收紧
2. 握距略宽于肩，杠铃在锁骨位置
3. 垂直向上推起，头部稍后让位
4. 控制下放至锁骨

**常见错误**：
- 腰部过度后仰 → 收紧核心和臀部
- 颈前推 → 头部让位，垂直推起
- 耸肩 → 保持肩部下沉

---

### 侧平举 (Lateral Raise)
**目标肌群**：三角肌中束

**执行步骤**：
1. 站立或坐姿，手持哑铃于身体两侧
2. 肘部微屈，向两侧抬起
3. 抬至肩高或略低
4. 控制下放

**常见错误**：
- 耸肩借力 → 减轻重量，专注中束
- 甩动身体 → 保持稳定
- 抬得过高 → 肩高即可

---

## 手臂动作

### 二头弯举 (Bicep Curl)
**目标肌群**：肱二头肌、肱肌

**执行步骤**：
1. 站立或坐姿，手持哑铃
2. 肘部固定于身体两侧
3. 弯举至顶峰收缩
4. 控制下放至手臂伸直

**常见错误**：
- 身体晃动借力 → 减轻重量
- 肘部前移 → 保持肘部固定
- 不完全下放 → 全程活动范围

**变式**：锤式弯举（侧重肱肌）、牧师凳弯举

---

### 三头下压 (Tricep Pushdown)
**目标肌群**：肱三头肌

**执行步骤**：
1. 站立，双手握住绳索或直杆
2. 肘部固定于身体两侧
3. 向下压至手臂伸直
4. 控制回到起始位置

**常见错误**：
- 肘部外展 → 保持贴紧身体
- 身体前倾借力 → 保持躯干稳定
- 手腕弯曲 → 保持手腕中立

---

## 核心动作

### 平板支撑 (Plank)
**目标肌群**：腹横肌、腹直肌、核心稳定性

**执行步骤**：
1. 前臂支撑，身体呈直线
2. 核心收紧，臀部不塌陷不上翘
3. 保持呼吸，不要憋气
4. 维持至目标时间

**常见错误**：
- 腰部下沉 → 收紧核心和臀部
- 臀部过高 → 调整至直线
- 憋气 → 保持正常呼吸

**进阶**：单臂平板、平板支撑抬腿、动态平板

---

### 悬垂举腿 (Hanging Leg Raise)
**目标肌群**：腹直肌、髋屈肌

**执行步骤**：
1. 悬挂于单杠，手臂伸直
2. 核心收紧，抬起双腿
3. 尽量抬至水平或更高
4. 控制下放

**常见错误**：
- 摆动借力 → 控制动作
- 只抬膝盖 → 尽量伸直腿
- 下落太快 → 离心控制

**退阶**：仰卧举腿、器械辅助

---

## 动作选择原则

### 新手优先学习
1. 深蹲、硬拉、卧推、肩推、划船、引体向上
2. 这些复合动作建立基础力量和动作模式

### 设备受限时的替代
| 健身房动作 | 家庭替代 |
|-----------|---------|
| 卧推 | 俯卧撑 |
| 深蹲 | 自重深蹲/弓步蹲 |
| 硬拉 | 单腿硬拉（用水瓶） |
| 划船 | 弹力带划船/毛巾划船 |
| 肩推 |  Pike 俯卧撑 |
| 引体向上 | 门框划船/弹力带下拉 |

### 伤病调整
- **膝盖问题**：减少深蹲深度，改用腿举
- **腰部问题**：避免大重量硬拉，改用器械
- **肩部问题**：避免颈后推举，改用哑铃
- **手腕问题**：使用护腕，改用中立握
