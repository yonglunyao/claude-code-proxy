# 训练计划模板 (Plan Templates)

## 新手计划 (0-6 个月经验)

### 全身训练 A/B 模式

**训练频率**：每周 3 次（周一/三/五 或 隔天）

**训练 A**：
| 动作 | 组数 | 次数 | 组间休息 | 目标肌群 |
|-----|------|------|---------|---------|
| 深蹲/腿举 | 3 | 8-12 | 90 秒 | 腿部 |
| 卧推/俯卧撑 | 3 | 8-12 | 90 秒 | 胸部/三头 |
| 划船/下拉 | 3 | 8-12 | 90 秒 | 背部/二头 |
| 肩推 | 2 | 10-15 | 60 秒 | 肩部 |
| 平板支撑 | 2 | 30-60 秒 | 60 秒 | 核心 |

**训练 B**：
| 动作 | 组数 | 次数 | 组间休息 | 目标肌群 |
|-----|------|------|---------|---------|
| 硬拉/罗马尼亚硬拉 | 3 | 6-10 | 120 秒 | 后链 |
| 上斜卧推/哑铃推举 | 3 | 8-12 | 90 秒 | 上胸 |
| 引体向上/高位下拉 | 3 | 6-10 | 90 秒 | 背阔肌 |
| 侧平举 | 2 | 12-15 | 60 秒 | 中束 |
| 卷腹 | 2 | 15-20 | 60 秒 | 腹肌 |

**执行方式**：A-B-A 周 / B-A-B 周交替

## 高级计划 (18+ 个月经验)

### 兄弟分化 (Bro Split)

**训练频率**：每周 5-6 次

| 周几 | 训练部位 | 容量 |
|-----|---------|------|
| 周一 | 胸部 | 高 (16-20 组) |
| 周二 | 背部 | 高 (16-20 组) |
| 周三 | 腿部 | 高 (18-22 组) |
| 周四 | 肩部 | 中 (12-16 组) |
| 周五 | 手臂 | 中 (12-16 组) |
| 周六 | 弱项/有氧 | 低 |
| 周日 | 休息 | - |

### 力量举专项

**训练频率**：每周 4-5 次，围绕三大项

**主项日**：
- 深蹲日：深蹲主项 + 辅助
- 卧推日：卧推主项 + 辅助
- 硬拉日：硬拉主项 + 辅助

**辅助日**：
- 技术日：轻重量技术训练
- 弱点日：针对弱项肌群

## 家庭自重训练

### 无设备全身训练

**训练频率**：每周 3-4 次

| 动作 | 组数 | 次数 | 进阶方式 |
|-----|------|------|---------|
| 深蹲 | 3 | 15-25 | 单腿深蹲 |
| 俯卧撑 | 3 | 10-20 | 钻石/弓箭手 |
| 反向划船 | 3 | 8-15 | 抬高脚 |
| 弓步蹲 | 3 | 12-20/腿 | 跳跃弓步 |
| 平板支撑 | 3 | 45-90 秒 | 单臂平板 |
| 波比跳 | 3 | 10-15 | - |

---

## 减脂专项计划

### 高强度间歇 + 力量

**训练频率**：每周 4-5 次

| 类型 | 频率 | 时长 |
|-----|------|------|
| 力量训练 | 3 次 | 45-60 分钟 |
| HIIT | 2 次 | 20-30 分钟 |
| LISS 有氧 | 1-2 次 | 30-45 分钟 |

**力量训练**：保持大重量，防止肌肉流失
**HIIT 示例**：30 秒冲刺 / 90 秒慢走 × 8 轮

---

## 增肌专项计划

### 高容量训练

**训练频率**：每周 5-6 次

**关键原则**：
- 每肌群每周 10-20 组
- 次数范围 6-15 次
- RPE 7-9（保留 1-3 次力竭余量）
- 渐进超负荷每周记录

---

## 计划选择指南

| 用户情况 | 推荐计划 | 理由 |
|---------|---------|------|
| 纯新手，时间少 | 全身 A/B | 频率高，学习快 |
| 新手，有时间 | 全身 A/B | 打好基础 |
| 6 个月 + 经验 | 上下肢分化 | 容量适中 |
| 想最大化增肌 | PPL | 高频率高容量 |
| 时间不固定 | 全身或上下肢 | 灵活调整 |
| 家庭训练 | 自重计划 | 无需设备 |
| 主要减脂 | 力量+HIIT | 保留肌肉 |
| 主要增肌 | PPL/兄弟分化 | 高容量 |
| 力量举爱好者 | 力量举专项 | 专项化 |
