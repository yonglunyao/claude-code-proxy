# Blood Sugar 血糖健康

---

## 一、数据获取策略

### 单指标查询 → 只取对应工具

| 用户意图 | 工具 |
|---------|------|
| "我血糖多少" / "今天血糖" | `get_blood_glucose --date today` |
| "最近血糖趋势" | `get_blood_glucose --last-days 7` |
| "饮食记录" / "今天吃了什么" | `get_nutrition --date today` |

### 综合分析 → 按场景并行取数

**血糖 + 饮食联合分析**
```
get_blood_glucose --date today
get_nutrition     --date today
```

**趋势分析（近期血糖 + 饮食）**
```
get_blood_glucose --last-days 7
get_nutrition     --last-days 7
```

## 二、分析框架

### 2.1 血糖分级

| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|-----------|--------|
| 空腹血糖（mmol/L） | 3.9–6.1 | 6.1–7.0 | ≥ 7.0 |
| 餐后 2 小时（mmol/L） | < 7.8 | 7.8–11.1 | ≥ 11.1 |

**核心原则**：糖尿病前期通过生活方式改变是**可逆的**，这是关键的干预窗口期。

### 2.2 控糖策略

**进食顺序效应（有循证支持）**：先蔬菜 → 再蛋白质和脂肪 → 最后碳水，可减少餐后血糖飙升 20–30%。

**餐后步行**（最有效的单一干预）：餐后 30 分钟内步行 15–30 分钟，可减少餐后血糖飙升 1–2 mmol/L，即使 10 分钟也有效。

**进餐规律**：规律进餐时间有助于稳定血糖；避免跳餐（引发代偿性暴食和更大血糖波动）；深夜进食血糖反应更高（昼夜节律性胰岛素抵抗）。

### 2.3 升糖指数（GI）参考

| GI 类别 | 范围 | 示例 |
|---------|------|------|
| 低 GI | < 55 | 燕麦、糙米、豆类、大多数蔬菜、苹果 |
| 中 GI | 55–70 | 全麦面包、甜玉米、香蕉 |
| 高 GI | > 70 | 白米饭、白面包、土豆泥、含糖饮料 |

> GI 受搭配食物、烹饪方式、食用量及个体差异影响，血糖负荷比单独 GI 更重要。

### 2.4 血糖 + 营养联合分析

将营养数据与血糖读数对应，识别：
- **问题食物**：哪顿饭后血糖飙升明显
- **安全食物**：哪些食物组合血糖平稳
- **碳水总量**：当日碳水是否过高导致持续偏高

### 2.5 低血糖识别

| 血糖水平 | 严重程度 | 症状 |
|----------|---------|------|
| 3.9–3.3 mmol/L | 轻度 | 颤抖、出汗、饥饿、焦虑 |
| 3.3–2.8 mmol/L | 中度 | 意识混乱、视力模糊 |
| < 2.8 mmol/L | 重度 | 意识丧失 — 紧急情况 |

**15 法则**（轻中度低血糖）：摄入 15g 快速碳水，等待 15 分钟，复查血糖。

### 2.6 跨域分析

- **血糖 + 运动**：有氧运动即时降糖；抗阻训练改善长期胰岛素敏感性
- **血糖 + 睡眠**：睡眠不足（< 6 小时）显著增加胰岛素抵抗，即使 1–2 天也有影响
- **血糖 + 压力**：长期压力 → 皮质醇 → 空腹血糖升高
- **血糖 + 体重**：内脏脂肪是胰岛素抵抗最强预测因素，减重 5% 即可显著改善血糖

## 三、红线阈值

| 触发条件 | 级别 |
|---------|------|
| 空腹血糖持续 ≥ 7.0 mmol/L | 软红线（建议就医评估糖尿病诊断） |
| 餐后血糖 > 11.1 mmol/L | 软红线（糖尿病范围，建议就医） |
| 低血糖症状（颤抖、意识混乱） | 软红线（立即摄入快速糖分；15 分钟未改善则就医） |
| 重度低血糖（< 2.8 或意识丧失） | 硬红线（立即拨打急救电话） |
| 询问调整或停用降糖药 | 边界（药物调整必须由医生决定，明确拒绝给出建议） |
