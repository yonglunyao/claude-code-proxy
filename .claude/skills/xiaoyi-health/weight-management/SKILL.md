# Weight Management 体重与体成分

---

## 一、数据获取策略

### 单指标查询 → 只取对应工具

| 用户意图 | 工具 |
|---------|------|
| "我今天体重多少" / "BMI 是多少" / "体脂率" | `get_body_composition --date today` |
| "最近体重趋势" / "这个月体重变化" | `get_body_composition --last-days 30` |

### 综合分析 → 按场景并行取数

**进度检查（体重 + 运动）**
```
get_body_composition --last-days 30
get_workouts         --last-days 30
```

**平台期诊断（体重 + 饮食 + 运动）**
```
get_body_composition --last-days 30
get_nutrition        --last-days 14
get_workouts         --last-days 14
```

**体重波动（女性用户，结合经期）**
```
get_body_composition  --last-days 14
get_menstrual_cycle   --date today
```

## 二、分析框架

### 2.1 BMI 参考范围

| 分类 | BMI |
|------|-----|
| 偏瘦 | < 18.5 |
| 正常 | 18.5–23.9 |
| 超重 | 24.0–27.9 |
| 肥胖 | ≥ 28.0 |

### 2.2 体成分参考标准

| 指标 | 健康范围 |
|------|---------|
| 体脂率（男性 18–39 岁） | 10–20% |
| 体脂率（男性 40–59 岁） | 12–22% |
| 体脂率（男性 60 岁以上） | 14–24% |
| 体脂率（女性 18–39 岁） | 18–28% |
| 体脂率（女性 40–59 岁） | 20–32% |
| 体脂率（女性 60 岁以上） | 22–34% |
| 健康减重速率 | 0.5–1.0 kg/周（不超过体重的 1%） |
| 热量缺口 | 500–750 kcal/天（不超过 1,000 kcal） |
| 最低热量摄入 | 女性 ≥ 1,200 kcal，男性 ≥ 1,500 kcal |
| 减脂期蛋白质 | 1.6–2.2 g/kg 体重/天 |

### 2.3 进度评估

关注**体脂率和肌肉量变化趋势**，而非只看体重数字：

| 场景 | 含义 |
|------|------|
| 体重下降、体脂下降、肌肉维持 | 理想减脂状态 |
| 体重稳定、体脂下降、肌肉增加 | 身体重塑，非常好 |
| 体重快速下降（> 1.5 kg/周） | 可能流失肌肉，需放慢节奏 |
| 体重每日波动 ±1 kg | 正常水分/钠盐波动，可忽略 |
| 体重停滞 10 天以上 | 真正的平台期，需调查能量平衡 |

### 2.4 平台期诊断

当体重停滞 10 天以上：
1. 检查热量摄入是否悄悄增加（饮食疲劳）
2. 检查 NEAT 是否下降（身体适应）
3. 建议：1 周维持热量饮食恢复代谢，再恢复热量缺口

### 2.5 女性月经周期注意事项

- 经前体重增加 0.5–2 kg 是水分潴留，**不是脂肪增长**
- 月经来潮后体重通常回落
- 最好在同一月经周期阶段进行月度对比
- 黄体期食欲增加是激素作用，不是缺乏意志力

### 2.6 跨域分析

- **体重 + 运动**：力量训练在减脂期保留肌肉比额外有氧更重要；运动后体重增加通常是糖原 + 水分
- **体重 + 睡眠**：睡眠不足（< 6 小时）→ 饥饿激素升高 + 饱腹感激素下降 → 难以控制食欲
- **体重 + 压力**：慢性压力 → 皮质醇升高 → 促进腹部脂肪堆积

### 2.7 沟通原则（体重领域特有）

- **重身体成分，轻体重数字**：不要只报告体重，要解读体脂率和肌肉量变化
- **体重焦虑**：不使用评判性语言，将正常波动解释清楚
- **平台期鼓励**："平台期意味着身体已经适应了，这是健康代谢的标志，调整策略即可"

## 三、红线阈值

| 触发条件 | 级别 |
|---------|------|
| 每周减重 > 2 kg | 软红线（警告肌肉流失和代谢损伤风险，建议咨询营养师） |
| 每日热量 < 1,000 kcal | 软红线（低于安全最低限，建议就医） |
| 进食障碍迹象（强迫记录、负罪感、暴食-节食循环） | 软红线（温暖无评判，建议寻求专业帮助） |
| BMI < 18.5 仍想减重 | 软红线（建议与医生讨论目标） |
