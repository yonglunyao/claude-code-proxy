# Fitness 运动与体能

---

## 概念说明（重要）

**workouts vs activity**

| | `get_workouts` | `get_activity_data` |
|---|---|---|
| 触发方式 | 用户主动开启/结束运动记录 | 系统全天自动采集 |
| 覆盖范围 | 单次运动记录（跑步、骑行等） | 所有日常活动（含家务、爬楼等） |
| 热量含义 | 该次运动消耗的卡路里 | 全天所有活动消耗的总卡路里 |

**运动时长 vs 锻炼时长**
- **运动时长**（workouts.durationMinutes）：用户主动开启到结束记录的完整时间
- **锻炼时长**（activity.activeMinutes）：心率达中高强度的时间自动累积，无需开启记录

两者可重叠但不等同：未开启记录的快步走计入锻炼时长，不出现在运动记录中。

## 一、数据获取策略

### 单指标查询 → 只取对应工具

| 用户意图 | 工具 |
|---------|------|
| "今天运动了什么" / "跑步/健身记录" | `get_workouts` |
| "今天走了多少步" / "活动量" / "锻炼时长" | `get_activity_data` |
| "VO2Max" / "有氧能力" / "心肺功能" | `get_vo2max` |
| "恢复状态" / "今天该不该练" | `get_hrv` + `get_resting_heart_rate` |

### 综合分析 → 按场景并行取数

**今日运动全貌**
```
get_workouts      --date today
get_activity_data --date today
```

**恢复状态**
```
get_hrv                --last-days 7
get_resting_heart_rate --last-days 7
```

**体能趋势（长期）**
```
get_vo2max   --last-days 30
get_workouts --last-days 30
```

## 二、分析框架

### 2.1 日常活动评估

**步数**：与用户自身目标（`stepsGoal`）和近期历史均值对比。

**锻炼时长（activeMinutes）目标（WHO）**：每周 ≥ 150 分钟中等强度，或 ≥ 75 分钟高强度；另加每周 2 天肌肉强化活动。

**久坐风险**：`activeHours` < 6 提示大部分时间处于静止状态，建议每小时起身活动。

### 2.2 运动记录分析

**训练强度判断**（通过 `avgHeartRate`，需用户年龄估算最大心率：220 − 年龄，无年龄时主动询问）：

| 平均心率占比 | 区间 | 目的 |
|------------|------|------|
| 60–70% | 有氧燃脂 | 减脂，长时间可持续 |
| 70–80% | 有氧耐力 | 提升心肺，每周 2–3 次 |
| 80–90% | 乳酸阈值 | 高强度，需充分恢复 |
| > 90% | 最大摄氧量区间 | 短时冲刺，谨慎安排 |

**80/20 原则**：80% 训练在燃脂 / 耐力区，20% 为高强度。

### 2.3 恢复评估（HRV + 静息心率）

| 状态 | 信号 | 建议 |
|------|------|------|
| 绿灯 | HRV ≥ 基线，静息心率正常 | 按计划训练 |
| 黄灯 | HRV 低 5–15%，静息心率略升 | 降低强度或主动恢复（散步、拉伸） |
| 红灯 | HRV 低 > 15%，静息心率高 > 5 bpm | 安排完全恢复，不进行高强度训练 |
| 过度训练警告 | HRV 低 > 25%，静息心率持续升高 | 停训 2–3 天 |

**运动后心率恢复（HRR）**：HRR = 运动结束即时心率 − 1 分钟后心率

| HRR | 评级 |
|-----|------|
| ≥ 20 bpm | 优秀 |
| 12–20 bpm | 正常 |
| < 12 bpm | 偏低，关注睡眠质量与训练节奏 |

**常见关联**：
- 运动后 48 小时静息心率仍高于基线 → 可能过度训练
- 睡眠不足（< 6 小时）→ 次日静息心率升高 3–8 bpm，影响训练准备度

### 2.4 VO2Max 体能趋势

VO2Max 是有氧运动能力和心肺功能的核心长期指标，数值越高代表有氧能力越强。

- 变化缓慢，需 30–90 天数据观察趋势，持续训练 6–8 周才能看到明显提升
- 优先关注**趋势方向**（上升 / 平稳 / 下降），而非与人群标准比较
- 静息心率下降 + VO2Max 上升 = 心肺功能改善的双重信号

### 2.5 运动中血氧监测

运动时 SpO2 正常应维持在 ≥ 95%。

| SpO2 | 信号 | 建议 |
|------|------|------|
| ≥ 95% | 正常 | 继续当前强度 |
| 92–94% | 轻度下降 | 降低强度，关注呼吸 |
| < 92% | 异常 | 停止运动，若持续建议就医 |

影响因素：高海拔、过度训练、呼吸道感染会导致运动中 SpO2 下降。运动后应在 5 分钟内恢复至 ≥ 96%。

## 三、红线阈值

| 触发条件 | 级别 |
|---------|------|
| 运动中出现胸痛、头晕或异常气短 | 硬红线（立即停止，就医前勿再训练） |
| 每天 2 小时以上高强度训练无休息日、使用自我惩罚式语言 | 软红线（温和询问感受，强调恢复同样重要） |
| 伤后想立即恢复训练 | 软红线（建议先获得医生许可） |
