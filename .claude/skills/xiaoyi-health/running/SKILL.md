# Running 跑步专项分析

---

## 一、数据获取策略

**单次跑步分析**
```
get_workouts         --date today          # 或指定日期
get_hrv              --last-days 3
get_resting_heart_rate --last-days 3
```

**训练趋势分析（近 4 周）**
```
get_workouts         --last-days 28
get_hrv              --last-days 28
get_resting_heart_rate --last-days 28
```

> 步频、触地时间、垂直振幅、触地平衡等高级指标依赖设备支持，数据中不存在时跳过对应分析。
>
> 心率区间计算需要用户年龄（最大心率 = 220 − 年龄）；未知时主动询问，或使用通用参考值并说明局限性。

---

## 二、分析框架

### 2.1 心率训练区间

**基于最大心率百分比（%HRmax）**

| 区间 | %HRmax | 特征 | 适用训练类型 |
|------|--------|------|------------|
| 热身 | 50–60% | 基本体能 / 热身 | 快走 |
| 燃脂 | 60–70% | 脂肪供能为主 | 慢跑 |
| 有氧耐力 | 70–80% | 提升肺活量和有氧能力 | 轻松跑 |
| 无氧耐力 | 80–90% | 速度提升，需控制时间 | 节奏跑 / 间歇跑 |
| 极限 | 90–100% | 乳酸快速积累，严格控制 | 极速跑 |

**基于储备心率百分比（%HRR，HRR = 最大心率 − 静息心率）**

| 区间 | %HRR | 特征 |
|------|------|------|
| 有氧基础 | 59–74% | 轻松可对话，培养有氧基础 |
| 有氧进阶 | 74–84% | 呼吸加重但不喘，模拟马拉松配速 |
| 乳酸阈值 | 84–88% | 吃力，乳酸堆积，提升阈值能力 |
| 无氧基础 | 88–95% | 很吃力，配合间歇模式 |
| 无氧进阶 | 95–100% | 极度吃力，高强度间歇 |

> 有静息心率数据时优先使用储备心率法，更个性化；无静息心率时使用最大心率百分比法。

### 2.2 跑步技术参考区间

| 指标 | 参考标准 |
|------|---------|
| **步频** | 普通跑者 160–170 步/分；进阶跑者 175–185 步/分 |
| **触地时间** | 优秀运动员 ≤ 220ms；数值越低跑步效率越高 |
| **左右触地平衡（左侧占比）** | 49.5–50.5%：均衡；48.5–49.5%：右侧略偏长；< 48.5%：右侧严重偏长；50.5–51.5%：左侧略偏长；> 51.5%：左侧严重偏长 |
| **垂直振幅** | ≤ 5.5cm：优秀；5.5–8cm：良好；8–12cm：正常；12–15cm：发展中；> 15cm：待优化 |

步频低 + 步幅大 = 受伤风险高，优先建议提升步频而非步幅。

### 2.3 训练负荷管理

**10% 法则**：每周总跑量增幅 ≤ 10%，防止过度训练损伤。

**配速类型分布（80/20 原则）**：
- 轻松跑（有氧基础区间）占全周训练量约 80%
- 节奏跑 / 间歇跑（乳酸阈值及以上）占约 20%

**恢复评估**：结合 HRV + 静息心率（参考 fitness skill 中的恢复评估框架）

### 2.4 输出结构

完整跑步分析报告按以下顺序输出：

1. **运动概览** — 距离、时长、配速、消耗热量等基本数据
2. **运动表现深度解析**
   - 心率与训练强度分析（各区间时间占比，评估心肺负荷）
   - 跑步技术评估（步频、步幅、触地时间、垂直振幅、触地平衡，仅在有数据时）
   - 配速策略分析（各阶段配速分布，负距跑 / 均速 / 正距跑判断）
   - 训练负荷与恢复评估（结合 HRV + 静息心率）
   - 近期训练趋势（仅趋势分析场景）
3. **综合评估与建议** — 优势总结、主要问题、针对性训练建议
4. **个性化训练计划示例**
   - 短期调整（未来一周）
   - 中期优化（未来 4 周）
5. **安全提醒** — 补水、营养、热身、伤害预防
6. **报告局限性** — 指出缺失数据或设备精度限制，建议用户补充信息

> 不生成数据图表（图表语法禁用，见 healthy-shared）。每条建议必须引用用户的具体数据，禁止笼统建议。

## 三、红线阈值

| 触发条件 | 级别 |
|---------|------|
| 每周跑量增幅 > 30% 且 HRV 持续下降 | 软红线（过度训练风险，建议减量并安排恢复日） |
| 跑步中出现胸痛、头晕、呼吸困难 | 硬红线（立即停止，就医前勿再训练） |
| 触地平衡严重不对称（左侧或右侧 < 48.5% 或 > 51.5%）且持续多次 | 软红线（步态问题，建议关注并考虑跑步姿势评估） |
| 伤后急于恢复训练 | 软红线（建议先获得医生许可再恢复） |
